# 健身档案

> 静态档案，低频更新。训练日志见 [log.md](log.md)

## 🎯 目标

- **主目标**: 干净增肌至 80kg 熔断线
- **专项目标**: 双板滑雪专项体能储备（抗侧倾、核心稳定、股四力量）

---

## 📐 训练体系

| 维度 | 设定 |
|------|------|
| **分化** | PPL 变体 → 四模板轮转：拉 → 推A(胸) → 腿 → 推B(肩) |
| **核心动作次数区间** | 6-10RM（机械张力区间），腿日可下探至 5-6RM |
| **训练频率** | 每周 3-4 天 |
| **单次时长** | 60-90 分钟 |
| **场地** | 健身房，自由重量 + 固定器械 |
| **休息规则** | 休息不跳项，轮到哪做哪 |

---

## 🍽️ 饮食

- **模式**: Lean Bulk（干净增肌）
- **热量**: 微盈余 200-300 kcal/天
- **蛋白质**: ≥150g/天
- **碳水**: 动态调度（训练日偏高，休息日偏低）
- **补剂**: 肌酸日常补充

---

## 📏 体测

- **工具**: JP-3 体脂钳（不依赖体脂秤）
- **基准**（2026-05-11）:
  - 体重: 74.4kg
  - 腹部: 18mm
  - 胸部: 12.6mm
  - 大腿: 10.2mm

---

## ⚠️ 伤病与限制

- **引体向上/悬垂举腿**: 避免重度悬垂拉扯
- **宽站距深蹲**: 微调避免髋屈挤压
- **胸骨老伤**: 卧推不过度撑开胸廓
- **外展易抽筋**: 需充分热身 + 电解质补充

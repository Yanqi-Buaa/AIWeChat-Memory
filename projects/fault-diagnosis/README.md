# 故障诊断项目

## 项目信息

- **方向**：基于振动信号的结构故障诊断
- **技术路径**：时序信号处理 → 三维频谱图像处理 → sim2real 迁移学习
- **框架**：PyTorch（默认参考资料：李沐《动手学深度学习》D2L）
- **数据集**：私有数据集（待补充细节）
- **当前目标**：在私有数据集上建立合理 baseline
- **时间线**：无硬性截止日期

## 协作模式

- 用户（博士生）审核所有任务后再布置给师弟
- 助手负责：学习路径规划、任务分解、进度管理、卡点分析
- 决策权在用户

## 目录文件

- `junior-profile.md` — 师弟档案（背景、知识状态、技能树）
- `tasks.md` — 任务池（待分配/进行中/已完成）
- `log.md` — 进展日志（时间线记录）

## 看门狗故障排查与修复（21:15~）
- **发现**：看门狗任务一次都没触发过，Last Run Time = 1999/11/30，错误码 267011
- **根因**：`process.py _create_watchdog_task()` 中 `/st "00:00:30"` 被误当作"30秒后"的延迟参数，实际 `/st` 是绝对时钟时间，`00:00:30` = 凌晨12:00:30
- **修复**：改用 `time.localtime() + 30s` 计算绝对时间，同时传入 `/sd`（日期）和 `/st`（时间）
- **清理**：删除了残留的过期看门狗任务
